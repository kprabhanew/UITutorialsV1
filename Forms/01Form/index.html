AngularJS expands the standard HTML input elements by adding new features and taking care of the underlying model updates to reflect the changes in the input items.

  • the "ng-model"" attribute that lets us bind the input value to a model variable
  • we need model variables associated to the input elements for the validation to work
  • We'll see that the model variables bound to them will be null whenever their values are invalid.
  • This is a special check made by AngularJS to make sure that the model always contains only valid values (e.g. if you type a string that is not a number in the numeric field, then the model variable bound to it will be null).
